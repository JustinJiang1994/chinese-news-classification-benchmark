# 中文新闻分类项目

## 项目概述

本项目基于中文新闻数据集实现多分类任务，使用多种机器学习算法进行文本分类，并比较不同模型的效果。

## 项目结构

```
news_classifier/
├── data/                    # 数据文件
│   ├── car_news.csv         # 汽车新闻
│   ├── entertainment_news.csv # 娱乐新闻
│   ├── finance_news.csv     # 财经新闻
│   ├── home_news.csv        # 家居新闻

│   ├── technology_news.csv  # 科技新闻
│   └── stopwords.txt        # 停用词表
```

## 模型特点分析

### 整体性能排名
1. **逻辑回归**: 69.43% (最佳)
2. **朴素贝叶斯**: 68.34%
3. **SVM**: 67.42%
4. **随机森林**: 62.07%

### 各类别表现分析

#### 表现最好的类别
- **体育**: 所有模型都表现很好，逻辑回归最佳(84.61%)
- **房产**: 所有模型都表现良好，逻辑回归最佳(81.28%)
- **汽车**: 所有模型都表现良好，逻辑回归最佳(81.39%)

#### 表现较差的类别
- **社会**: 所有模型都表现较差，逻辑回归相对较好(47.86%)
- **财经**: 所有模型都表现较差，逻辑回归相对较好(51.11%)
- **家居**: 所有模型都表现较差，逻辑回归相对较好(55.38%)

### 模型特点总结

1. **逻辑回归**: 
   - 整体性能最佳，训练速度中等
   - 对大多数类别都有较好的表现
   - 适合作为基准模型

2. **朴素贝叶斯**: 
   - 训练速度最快，性能稳定
   - 对专业术语较多的类别表现较好
   - 适合快速原型开发

3. **SVM**: 
   - 训练时间最长，但性能稳定
   - 对某些类别（如体育）有特殊优势
   - 适合对性能要求较高的场景

4. **随机森林**: 
   - 性能相对较低，但可解释性强
   - 提供了特征重要性分析
   - 适合需要模型解释的场景

### 特征重要性分析（随机森林）
随机森林模型提供了特征重要性分析，可以帮助理解哪些词汇对分类最重要。主要重要特征包括：
- 各类别的专业术语
- 高频词汇
- 具有区分性的词汇组合

# 中文新闻分类项目

## 项目概述

本项目基于中文新闻数据集实现多分类任务，使用多种机器学习算法进行文本分类，并比较不同模型的效果。

## 项目结构

```
news_classifier/
├── data/                    # 数据文件
│   ├── car_news.csv         # 汽车新闻
│   ├── entertainment_news.csv # 娱乐新闻
│   ├── finance_news.csv     # 财经新闻
│   ├── home_news.csv        # 家居新闻
│   ├── house_news.csv       # 房产新闻
│   ├── international_news.csv # 国际新闻
│   ├── military_news.csv    # 军事新闻
│   ├── society_news.csv     # 社会新闻
│   ├── sports_news.csv      # 体育新闻
│   ├── technology_news.csv  # 科技新闻
│   └── stopwords.txt        # 停用词表
├── src/                     # 源代码
│   ├── naive_bayes_classifier.py # 朴素贝叶斯分类器
│   └── svm_classifier.py    # SVM分类器
├── models/                  # 训练好的模型
│   ├── naive_bayes_model.pkl
│   └── svm_model.pkl
├── results/                 # 结果文件
│   ├── naive_bayes_results.txt
│   ├── naive_bayes_confusion_matrix.png
│   ├── svm_results.txt
│   └── svm_confusion_matrix.png
├── utils/                   # 工具脚本
│   └── analyze_data.py      # 数据分析脚本
├── run_naive_bayes.py       # 运行朴素贝叶斯分类器
├── run_svm.py              # 运行SVM分类器
├── test_naive_bayes.py      # 测试朴素贝叶斯分类器
├── test_svm.py             # 测试SVM分类器
└── requirements.txt         # 项目依赖
```

### 朴素贝叶斯分类器结果

#### 模型配置
- **算法**: 多项式朴素贝叶斯 (MultinomialNB)
- **特征提取**: TF-IDF向量化 (max_features=10000, ngram_range=(1,2))
- **文本预处理**: jieba分词 + 停用词过滤
- **数据平衡**: 每类限制5000条样本，总计50000条
- **训练/测试比例**: 8:2

## 模型特点分析

### 朴素贝叶斯 vs SVM 对比

1. **整体性能**: 朴素贝叶斯略优于SVM，准确率高出0.92个百分点
2. **训练速度**: 朴素贝叶斯训练速度更快（14秒 vs 79秒）
3. **类别表现差异**:
   - **朴素贝叶斯优势类别**: 房产、汽车、娱乐、国际
   - **SVM优势类别**: 体育、军事
   - **共同表现较差**: 社会、财经、家居

4. **模型特点**:
   - **朴素贝叶斯**: 训练快速，对专业术语较多的类别表现较好
   - **SVM**: 训练较慢，但对某些类别（如体育）有更好的区分能力
